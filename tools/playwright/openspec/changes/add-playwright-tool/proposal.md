# Proposal

## Summary

Add a dedicated Playwright tool to the `common-skills` repository for browser debugging and DOM inspection.

## Motivation

The repository needs a reusable browser automation utility that can inspect runtime DOM, save screenshots, and provide a stable base for future browser-oriented helpers.

## Scope

- Create a self-contained `tools/playwright` directory.
- Add a documented CLI for page and element inspection.
- Store tool requirements and workflow in a tool-local OpenSpec tree.
